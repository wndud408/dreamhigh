  2. 향수 배합 배워서 직접 만들기 : 향수를 좋아해서 나에게 가장 잘 맞는 향으로 향수를 만들고 싶어서

  3. 탈색 해보기 : 한번 쯤은 원하는 색으로 염색 해보고 싶다는 생각이 들어서

  4. 혼자 여행 가기 : 혼자 여행하면 더 많이 생각해보고 느끼는 바도 있을 것 같아서

  5. 태연/아이유 콘서트 가기 : 일단 너무 좋아하는 가수이기 때문에 라이브로 공연을 보고 싶어서

  6. 영어, 독일어 공부하기 : 언어를 많이 할 줄 알수록 여행 갔을 때에 더 도움될 것 같아서

  7. 취미 생활 가지기 : 일이나 공부 말고 내가 재미있게 할 수 있는 일을 찾고 싶어서

  8. 부모님 여행 보내드리기 : 두 분이 좋은 곳 가서 편히 놀다 오셨으면 좋겠어서

  9. 이민 가기 : 스페인에서 평화롭게 살고 싶어서
